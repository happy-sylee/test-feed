# ===================================================================
# 카테고리별 주제 키워드
# ===================================================================
# 형식:
#   [축:카테고리]  <- 대괄호로 카테고리 헤더 선언
#   키워드          <- 그 아래 한 줄에 하나씩
#
# ★ 필터링(수집) 규칙 [중요]:
#   논문이 피드에 뜨려면 아래 둘 중 하나를 만족해야 함:
#     (a) [SYSTEM] 축의 키워드가 하나라도 매칭  (예: lithium-ion, sodium-ion)
#     (b) [BATTERY] 섹션의 확정 키워드가 매칭   (예: battery, SEI, fast charging)
#   → COMPONENT(anode/cathode 등)만으로는 통과 못 함 (태깅에만 사용).
#   → 이렇게 하면 배터리와 무관한 전기화학/생물 논문이 걸러짐.
#
#   태깅(카테고리 표시)은 SYSTEM/COMPONENT 축으로 별도 수행.
#
# - 대소문자 구분 안 함 / '#' 주석 / 빈 줄 무시
# ===================================================================
